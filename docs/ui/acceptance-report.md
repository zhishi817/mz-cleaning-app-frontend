## 对比方法（人工验收）

建议使用图片查看器的“并排对比/透明叠加”功能，按以下项逐条确认：

- 顶部 Header：问候语字号/粗细、头像尺寸与右侧对齐
- 分段控件：圆角、阴影、选中态背景与文字颜色
- 日期卡片：卡片宽高、圆角、边框、选中态蓝色、底部圆点位置
- 区块标题：`今日任务` 与右侧 `3 个任务` 的字号/颜色/对齐
- 任务卡片整体：圆角、阴影、内边距、卡片间距
- 房源信息区：图片占位、标题、地址行图标与字体、右侧状态 pill
- 标签：`STUDIO` 灰色 pill 与 `早入住·优先` 红色 pill 的高度/圆角/图标间距
- 路线顺序：浅蓝背景、左侧图标底、右侧下拉框边框与文字
- 时间信息：两列排布、图标颜色、时间数字字号/粗细
- 密码卡片：三等分布局、蓝/绿/灰配色、字号与上下间距
- 指南链接：图标与文案颜色、对齐方式
- 底部 Tab：高度、图标大小、文字字号、选中态蓝色与未选中灰色

## 已实现的关键设计参数（用于复核）

- 主题主色：#2563EB
- 页面背景：#F6F7FB
- 卡片背景：#FFFFFF
- 边框：#EEF0F6（hairline）
- 字体权重：标题/关键数值使用 800–900
- 自适应：基于 375 宽度进行等比缩放（`src/lib/scale.ts`）

## 性能与体验检查

- 长内容使用 ScrollView，关闭垂直滚动条提升观感
- 关键数据使用 memo，避免不必要的重复渲染
- 使用 hairline 边框与轻量阴影，减少重绘成本
